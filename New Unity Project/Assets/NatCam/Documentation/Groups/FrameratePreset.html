FrameratePreset

enumeration in NatCamU.Core / Enumeration

Description

This contains presets that can be used for easily setting framerates across platforms and devices.



Static Variables

Default Default framerate. Usually 30FPS.
HighestFramerate Highest framerate. Defaults to 240FPS.
LowestFramerate Lowest framerate. Defaults to 15FPS.
SlowMotion Slow motion framerate. Defaults to 120FPS.
Smooth Smoother framerate. Defaults to 60FPS.
