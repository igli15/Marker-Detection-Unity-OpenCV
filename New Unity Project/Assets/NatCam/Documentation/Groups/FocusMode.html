FocusMode

enumeration in NatCamU.Core / Enumeration

Description

This contains modes for handling focusing.



Static Variables

AutoFocus The camera will focus automatically.
MacroFocus The camera will focus automatically on close-up objects.
Off Disable focusing.
SoftFocus The camera will focus automatically and softly.
TapToFocus The camera will only focus when SetFocus is called.
