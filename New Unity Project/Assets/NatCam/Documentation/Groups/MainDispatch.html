MainDispatch

class in NatCamU.Core.Utilities / inherits from IDispatch

Description

This is a dispatcher that executes its workload on the UI (main) thread. This is very useful for joining workload back to the main thread when Unity API calls are needed (because Unity is not thread-safe).



Public Functions

Dispatch Dispatch a delegate to be invoked.
Release Release the dispatcher and its resources.


Constructors

MainDispatch Creates a dispatcher that will execute on the main thread.
