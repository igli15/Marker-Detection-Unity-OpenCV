ConcurrentDispatch

class in NatCamU.Core.Utilities / inherits from IDispatch

Description

This is a dispatcher that executes its workload on a separate thread. This is very useful for performing long-running or computationally expensive tasks without freezing the UI (main) thread.



Public Functions

Dispatch Dispatch a delegate to be invoked.
Release Release the dispatcher and free its worker thread.


Constructors

ConcurrentDispatch Creates a dispatcher that will execute on a worker thread.
