ScaleMode

enumeration in NatCamU.Core / Enumeration

Description

This determines how NatCamScaler will scale the UI panel for the preview.



Static Variables

FillScreen Fill the screen however necessary.
FillView Fill the RectTransform however necessary.
FixedHeightVariableWidth Stretch the width, keep the height.
FixedWidthVariableHeight Stretch the height, keep the width.
None Do nothing.
