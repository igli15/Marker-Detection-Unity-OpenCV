SaveOrientation

enumeration in NatCamU.Extended / Enumeration

Description

This determines how photos are oriented when saved.



Static Variables

Rotation_0 Do not rotate.
Rotation_180 Rotate 180 degrees.
Rotation_270 Rotate 270 degrees clockwise.
Rotation_90 Rotate 90 degrees clockwise.
