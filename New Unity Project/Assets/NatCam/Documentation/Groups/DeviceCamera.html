DeviceCamera



Description

This class provides an abstraction for device cameras.



Public Functions

SetFocus Focus the camera to a viewport point.
SetFramerate Set the framerate.
SetFramerate Set the framerate.
SetPhotoResolution Set the photo resolution to one closest to the resolution preset.
SetPhotoResolution Set the photo resolution to one closest to (width, height).
SetPreviewResolution Set the preview resolution to one closest to the resolution preset.
SetPreviewResolution Set the preview resolution to one closest to (width, height).


Public Variables

ExposureBias Get or set the camera's exposure bias.
ExposureMode Get or set the camera's exposure mode.
Facing Get the facing of the camera.
FlashMode Get or set the camera's flash mode.
FocusMode Get or set the camera's focus mode.
HorizontalFOV The horizontal view angle of the camera in degrees.
IsFlashSupported Does the camera support flash?
IsTorchSupported Does the camera support torch?
IsZoomSupported Does the camera support zooming (ramped or smooth)?
MaxExposureBias The maximum exposure bias supported by the camera.
MinExposureBias The minimum exposure bias supported by the camera.
PhotoResolution Get the photo resolution of the camera.
PreviewResolution Get the preview resolution of the camera.
TorchMode Get or set the camera's torch mode.
VerticalFOV The vertical view angle of the camera in degrees.
ZoomRatio Get or set the camera's zoom ratio.


Static Variables

Cameras A readonly list of all cameras on the device.
FrontCamera The default front camera of this device.
RearCamera The default rear camera of this device.
