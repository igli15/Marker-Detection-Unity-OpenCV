BarcodeFormat

enumeration in NatCamU.Extended / Enumeration

Description

This contains formats of barcodes that NatCam can detect.



Static Variables

ALL Any and all barcodes.
CODE_128 Code 128.
CODE_39 Code 39.
CODE_93 Code 93.
DATA_MATRIX Data Matrix.
EAN_13 EAN 13 Code. ISBN codes fall into this category.
EAN_8 EAN 8 Code.
ITF Interleaved 2 of 5 Code.
PDF_417 PDF 417.
QR QR Code.
