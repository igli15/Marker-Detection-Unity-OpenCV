ResolutionPreset

enumeration in NatCamU.Core / Enumeration

Description

This contains presets that can be used for easily setting resolutions across platforms and devices.



Static Variables

FullHD Full HD. 1080p (1920x1080).
HD HD. 720p (1280x720).
HighestResolution Highest resolution the active camera supports.
LowestResolution Lowest resolution the active camera supports.
MediumResolution Medium resolution. Defaults to 640x480.
