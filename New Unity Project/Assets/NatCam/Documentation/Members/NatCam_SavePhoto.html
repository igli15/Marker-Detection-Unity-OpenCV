NatCam.SavePhoto

public static void SavePhoto (Texture2D photo, SaveMode mode, SaveOrientation orientation, SaveCallback callback);

Description

Save a photo to the device. 

NatCam.SavePhoto(
    texture,
    SaveMode.SaveToAppDocuments | SaveMode.SaveToPhotoGallery, // Save to app documents and device gallery
    SaveOrientation.Rotation_180, // Save upside down
    (mode, path) => Debug.Log("Photo saved with "+mode+" to path: "+path) // Log
);
        

NatCam.SavePhoto(texture);
