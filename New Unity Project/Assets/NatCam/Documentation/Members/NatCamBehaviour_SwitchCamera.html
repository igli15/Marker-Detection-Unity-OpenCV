NatCamBehaviour.SwitchCamera

public void SwitchCamera (int newCamera);

Description

Switch the active camera of the preview. An int or a DeviceCamera can be passed here. Note that the newCamera argument is optional. If nothing is passed in, NatCam will switch to the next available device camera.
 

//Switch to the front camera
SwitchCamera(DeviceCamera.FrontCamera);
        

//Switch to the next available camera
SwitchCamera();
        

//Switch to the default camera
SwitchCamera(0);
