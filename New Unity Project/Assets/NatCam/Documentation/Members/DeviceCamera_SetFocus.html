DeviceCamera.SetFocus

public void SetFocus (Vector2 viewportPoint);

Description

Focus the camera to a point corresponding to the provided Vector2 coordinates in viewport space (between [0,0] and [1,1]). 

// Focus at the screen centre
NatCam.Camera.SetFocus(new Vector2(0.5f, 0.5f));
