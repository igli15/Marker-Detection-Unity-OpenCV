NatCamMetadata.Decode

public static void Decode (Color32[] frame, int width, int height, IDispatch dispatch, MetadataCallback callback);

Description

Detect barcodes in a preview frame. Supply an INatCamDispatch which will be used to invoke the callback on the corresponding thread. The callback provided will be invoked for each barcode that has been detected.
 

INatCamDispatch mainDispatch; // Initialize this somewhere

void FindBarcodes () {
    NatCamMetadata.Decode(
        webCamTexture.GetPixels32(),
        webCamTexture.width,
        webCamTexture.height,
        mainDispatch,
        code => Debug.Log("Found barcode: "+code.ToString())
    );
}
