NatCam.CapturePhoto

public static void CapturePhoto (PhotoCallback callback);

Description

Capture a photo with the provided callback.

Please note that the photo returned through the callback MUST be freed when it it no longer in use. This is because Texture2D's are not automatically released when they go out of scope. Use Texture2D.Destroy(...). 

NatCam.CapturePhoto(OnPhotoCapture);

void OnPhotoCapture (Texture2D photo) {
    do stuff...
}
        

NatCam.CapturePhoto(photo => material.mainTexture = photo);
